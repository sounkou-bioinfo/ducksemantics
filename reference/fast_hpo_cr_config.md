# Show reticulate's active Python configuration

Show reticulate's active Python configuration

## Usage

``` r
fast_hpo_cr_config()
```

## Value

The object returned by
[`reticulate::py_config()`](https://rstudio.github.io/reticulate/reference/py_config.html).

## Examples

``` r
if (FALSE) { # interactive()
if (fast_hpo_cr_available()) {
  fast_hpo_cr_config()
}
}
```
