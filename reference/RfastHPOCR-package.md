# R bindings to `FastHPOCR`

`RfastHPOCR` provides a small R interface to the upstream Python
[`FastHPOCR`](https://github.com/tudorgroza/fast_hpo_cr) package using
[reticulate](https://rstudio.github.io/reticulate/reference/reticulate.html).
It exposes helpers for installing the Python dependency set, building
FastHPOCR ontology indexes, creating annotators, and converting
annotation objects to R data frames.
