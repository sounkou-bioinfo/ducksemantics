# Embedding matrix contract

S7 value object for a finite numeric embedding matrix with an expected
row count.

## Usage

``` r
DucksemanticsEmbeddingMatrix(embeddings = integer(0), rows = integer(0))
```

## Arguments

- embeddings:

  Numeric matrix.

- rows:

  Expected number of matrix rows.

## Value

A `DucksemanticsEmbeddingMatrix` object.
